# Wanted Front-end onBoarding # 1

## HayanMind Assignment : API와 연동하여 무한 스크롤(Infinite Scroll)을 구현

### 과제 구현 목록
- [x] ClassComponent 사용
- [x] SessionStorage 또는 LocalStorage 사용해서 이력을 관리
- [x] 외부 API를 사용하지 않고, Client의 리소스만 사용

### 설치 및 시작하는 법
```jsx
npm install && npm start
```
